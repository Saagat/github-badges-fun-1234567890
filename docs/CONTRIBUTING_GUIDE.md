# Contributing to Stellar Dev Suite

Thank you for your interest in contributing to the **Stellar Dev Suite**! We welcome contributions from the community to help make this project even better.

## How to Contribute

### 1. Reporting Bugs
- Use the **GitHub Issues** tab to report bugs.
- Include detailed steps to reproduce the issue and your environment details.

### 2. Suggesting Enhancements
- Open a new issue and label it as an `enhancement`.
- Describe the feature you'd like to see and why it would be useful.

### 3. Pull Requests
- Fork the repository.
- Create a new branch for your feature or fix.
- Ensure your code follows the project's aesthetic standards.
- Submit a Pull Request with a clear description of your changes.

## Code of Conduct
We are committed to providing a friendly, safe, and welcoming environment for all contributors. Please be respectful and professional in all interactions.
